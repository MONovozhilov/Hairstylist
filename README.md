Проект реализует пайплайн для анализа формы лица и генерации причесок с использованием AI.
Основные функции:
1. Сегментация волос на изображении с использованием модели на базе PSPNet
2. Классификация формы лица (овал, круг, квадрат, сердце, продолговатый)
3. Генерация новых причесок через Stable Diffusion Inpainting
4. Перенос прически с одного изображения на другое
