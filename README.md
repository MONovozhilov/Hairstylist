Проект реализует пайплайн для анализа формы лица и генерации причесок с использованием AI. Основные функции:
Сегментация волос на изображении с использованием модели на базе PSPNet
Классификация формы лица (овал, круг, квадрат, сердце, продолговатый)
Генерация новых причесок через Stable Diffusion Inpainting
Перенос прически с одного изображения на другое
